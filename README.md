## Koriscene tehnologije:
- React 19.2.5 + React Router 7.14.1 (jer nemamo bekend)
- JSX
- HTML
- CSS

- Vite kao alat za razvoj i buildovanje aplikacije (standardno za React aplikacije). 

## Koriscene resurse:
- https://fonts.google.com/
- https://www.svgrepo.com/vectors/

## Struktura projekta
Sav kod projekta se nalazi pod src folderu. Projekat prati komponentno orijentisan pristup
i najveci deo koda je napisan u JSX-u i u CSS-u. Takodje postoji dist (distribution) folder 
sa bundlovanom kodom koji se moze za koje treba server da ga servira. Ukoliko imate node 
mozete uci u ./dist i pokrenuti "npx serve -s".
